# Mac

Tips, shortcuts, and productivity tricks for macOS.

---

## Keyboard Shortcuts

### System

| Shortcut | Action |
|---|---|
| `CMD+~` | Switch between windows of the same app |
| `CMD+M` | Minimize window |
| `CMD+H` | Hide app |
| `CMD+Option+H` | Hide all other apps |
| `CMD+CTRL+F` | Toggle fullscreen |
| `CMD+SHIFT+3` | Screenshot entire screen |
| `CMD+SHIFT+4` | Screenshot selection |
| `CMD+SHIFT+4+Space` | Screenshot a window |


### Finder

| Shortcut | Action |
|---|---|
| `CMD+N` | New Finder window |
| `CMD+SHIFT+N` | New folder |
| `CMD+SHIFT+.` | Show/hide hidden files |
| `Space` | Quick Look preview |

## Miscellaneous

- **Force quit**: `CMD+Option+Esc` — opens Force Quit dialog
- **Emoji picker**: `CTRL+CMD+Space` anywhere
- **Character viewer**: Same as above, click the expand icon
- **AirDrop**: Works between Mac, iPhone, iPad over Bluetooth/WiFi — no setup needed
- **Handoff**: Continue tasks across Apple devices if signed into same Apple ID
- **Universal Clipboard**: Copy on iPhone, paste on Mac (and vice versa) with Handoff enabled
